## GUIAS DE COMPILACIÓN
### Windows
1. Instalar **MinGW** o usar **MSYS2**.  
2. Compilar con:  
   g++ archivo.cpp -o programa.exe
3. Ejecutar con:
   programa.exe

### Linux
1. Instalar g++:
   sudo apt-get install g++
2. Compilar con:
   g++ main.cpp juego_ai.cpp tablero.cpp juego.cpp -o main
3. Ejecutar con:
   ./main

### MacOS
1. Instalar compilador:
   xcode-select --install
2. Compilar con:
   g++ main.cpp juego_ai.cpp tablero.cpp juego.cpp -o main
3. Ejecutar con:
   ./main
